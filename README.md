# LumaPack
Pack to hack the 3DS in 11.15 with PicHaxx and NtrBoothax and USM in 11.16 (Picchaxx does not work in 11.16)

Includes : 
# Release
```
- LumaPack NTRBootHax (11.16)
- LumaPack PicHaxx (11.15)
- LumaPack USM (11.16)
```
